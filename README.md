🪨📄✂️ Rock Paper Scissors (React + Vite)

A modern and interactive Rock Paper Scissors game built with React and powered by Vite. Play against the computer, track your score, and enjoy a clean, minimal UI.

🚀 Live Demo

👉 https://rock-pape-scissor-play.netlify.app/

📸 Preview

✨ Features
🎮 Interactive gameplay (User vs Computer)
⚛️ Built using React (JSX components)
⚡ Powered by Vite for fast performance
🧠 Smart game logic with instant results
🏆 First to 10 wins system
🔁 Reset game functionality
📱 Fully responsive design
🧠 Game Logic
Choose between Rock, Paper, or Scissors
Computer randomly selects its move
Winner is decided:
🪨 Rock beats Scissors
✂️ Scissors beats Paper
📄 Paper beats Rock
Scores update dynamically until one reaches 10 wins
